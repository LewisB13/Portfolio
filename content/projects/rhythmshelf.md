RhythmShelf is a local Python application designed to clean, organise, and structure music libraries. It is built as a utility tool rather than a media player, focusing entirely on improving the quality and consistency of file organisation across large music collections.

The core problem RhythmShelf solves is the natural decay of music libraries over time. As files are added from different sources, duplicates accumulate, folder structures become inconsistent, and metadata tagging becomes incomplete or unreliable. RhythmShelf addresses this through a set of automated processing tools that operate directly on the file system.

The application includes duplicate detection and removal, allowing users to identify repeated tracks and safely clean their library. A flattening system restructures deeply nested directories into a simplified and consistent hierarchy, making the library easier to navigate and manage. In addition, a tagging system enables the application of metadata labels to music files, helping categorise and organise content based on user-defined rules or attributes.

RhythmShelf is designed to run locally in Python, giving users full control over their files without relying on external services or cloud-based systems. This makes it suitable for personal libraries, offline environments, and users who prefer direct filesystem control.

The focus of the project is automation and simplicity — reducing manual effort in maintaining a clean and usable music collection.

Future improvements may include rule-based organisation presets, smarter metadata inference, and a lightweight GUI built on top of the existing Python core.
