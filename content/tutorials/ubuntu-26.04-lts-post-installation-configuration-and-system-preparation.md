  ### Project Overview

  This project focused on the initial configuration and optimisation of a newly installed Ubuntu 26.04 LTS virtual machine running in VMware Workstation.

  The objective was to prepare the operating system for everyday use by applying system updates, enabling security features, installing essential utilities, and verifying system functionality through command-line administration.

  ### Technologies Used

  - Ubuntu 26.04 LTS
  - VMware Workstation
  - Linux Terminal (Bash)
  - UFW Firewall
  - Open VM Tools

  ### Tasks Completed

  - Updated package repositories and installed available system updates
  - Verified operating system and kernel versions
  - Reviewed CPU, memory and storage information
  - Installed Open VM Tools for improved VMware integration
  - Enabled and verified Ubuntu's firewall (UFW)
  - Installed common administration tools including Git, Curl and Nano
  - Created and managed directories and files using Linux commands
  - Performed basic system health and storage checks

  ### Skills Demonstrated

  - Linux Administration
  - Terminal Operations
  - Package Management
  - System Configuration
  - Virtual Machine Management
  - Security Hardening Fundamentals
  - Troubleshooting and Diagnostics

  ### Outcome

  A fully updated and properly configured Ubuntu 26.04 LTS virtual machine was prepared for future development, administration, networking and cybersecurity projects while demonstrating foundational Linux administration skills.
