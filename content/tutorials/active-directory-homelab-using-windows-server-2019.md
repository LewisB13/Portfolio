Built and configured a beginner-friendly **Active Directory homelab** environment using **Windows Server 2019** and multiple **Windows 10 client machines**.

The project involved deploying **Active Directory Domain Services (AD DS)**, configuring a domain controller, creating and managing user accounts, and joining Windows 10 systems to the domain environment.

### Key Skills Demonstrated

- Windows Server Administration
- Active Directory Domain Services (AD DS)
- User & Group Management
- Domain Controller Configuration
- Windows 10 Domain Joining
- Basic Networking & DNS
- IT Troubleshooting
- System Administration Fundamentals

### Technologies Used

- Windows Server 2019
- Windows 10
- Active Directory
- Oracle VirtualBox / VMware (if you used one)

### Project Outcome

Successfully created a functional Windows domain environment capable of centralized user management and client authentication, providing hands-on experience with core enterprise IT infrastructure concepts.
