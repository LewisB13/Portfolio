This project demonstrates the installation and configuration of **SearXNG**, a privacy-focused, self-hosted metasearch engine, on Windows 10 using Docker Desktop and WSL2.

The setup includes:

- Docker Compose configuration for container deployment
- WSL2 integration for Linux compatibility on Windows
- Local hosting of a private search engine
- Network configuration for optional LAN access
- Troubleshooting common Docker and port issues

This project highlights skills in:

- Docker & containerization
- Linux subsystem on Windows (WSL2)
- System configuration and networking
- Self-hosted web applications
- Privacy-focused software deployment
